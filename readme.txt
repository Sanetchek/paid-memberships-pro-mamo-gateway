Take credit card payments on your site using MAMO payment gateway.

== Description ==

This plugin integrates Paid Memberships Pro with MAMO payment gateway, enabling you to accept payments through payment links, subscriptions, and webhooks.

== Features ==

* Payment Links (standalone, modal, inline)
* Recurring payments via native subscriptions or MIT
* Webhook support for real-time payment notifications
* Refund support
* Sandbox/Production environment support
* Debug logging

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pmpro-mamo-gateway`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the gateway in Paid Memberships Pro → Settings → Payment Gateway

== Configuration ==

1. Get your API Key from MAMO Dashboard
2. Go to Paid Memberships Pro → Settings → Payment Gateway
3. Select "MAMO" as your gateway
4. Enter your API Key (Live and/or Sandbox)
5. Configure Webhook URL in MAMO dashboard
6. Enable logging if needed for debugging
